# Link Scraper

A Python tool that scrapes all links from any website and saves them to a file.

## What it does
- Takes any website URL as input
- Finds all links on that page
- Saves them to a timestamped .txt file

## How to use
1. Install requirements: pip install requests beautifulsoup4
2. Run: python scraper.py
3. Enter any website URL
4. Links saved automatically to a .txt file
